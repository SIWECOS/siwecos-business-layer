Die HSTS Header-Einstellung garantiert, dass der Verkehr zwischen Server und Client stets für die Kommunikation verschlüsselt werden muss. Sie teilt Webseiten-Besuchern mit, dass Ihre Webseite nur über eine verschlüsselte Verbindung (HTTPS) erreichbar ist und der Browser diese Einstellung für längere Zeit zwischenspeichern kann. Die Verbindung findet somit immer nur über eine sichere HTTPS-Verbindung und nie über das unsichere HTTP-Protokoll statt.
